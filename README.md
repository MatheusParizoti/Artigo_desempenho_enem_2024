# Análise de Desempenho dos Estudantes no ENEM 2024 - São Paulo

Este repositório contém a análise realizada sobre o desempenho dos estudantes do estado de São Paulo no Exame Nacional do Ensino Médio (ENEM) de 2024. A pesquisa investigou como fatores relacionados ao município e ao tipo de escola impactam nos resultados do exame.

## Estrutura do Repositório

- **`README.md`**: Este arquivo, que contém a descrição do projeto e as instruções para uso.
- **`código/`**: Pasta que contém os scripts e códigos utilizados para análise de dados.
- **`bases/`**: Pasta contendo as bases de dados utilizadas na pesquisa (disponibilizadas pelo INEP).
- **`certificado/`**: Pasta que contém o certificado comprovando a aprovação do artigo no congresso.

## Objetivo do Estudo

O objetivo deste estudo foi analisar os fatores que influenciam o desempenho dos estudantes do estado de São Paulo no ENEM 2024, considerando aspectos como:
- Tipo de escola (pública ou privada).
- Fatores socioeconômicos, como a renda familiar.
- Diferenças regionais e municipais.

## Metodologia

Utilizamos duas bases de dados disponibilizadas pelo INEP:
1. **Base de participantes**: Informações gerais sobre os inscritos no ENEM.
2. **Base de resultados**: Notas médias por disciplina e município de aplicação da prova.

A análise foi realizada utilizando a linguagem Python, com as bibliotecas `Pandas`, `Matplotlib`, `Seaborn`, `GeoPandas` e `Geobr` para tratamento de dados e visualização.

## Resultados

- **Escolas privadas** apresentaram desempenho superior em todas as disciplinas do ENEM 2024.
- **Renda familiar** mostrou correlação positiva com o desempenho, indicando que condições socioeconômicas influenciam os resultados.
- **Municípios como São João da Boa Vista, Valinhos e Ribeirão Preto** se destacaram com as maiores médias gerais, refletindo desigualdades regionais.
